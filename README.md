## Installation

### Installing software
```bash

sudo pacman -S zsh firefox vim gimp vscode rawtherapee kdenlive lmms mixxx yay

```

### Installing configs
```bash

git clone https://github.com/smiddy-001/arcolinux-dotfiles.git
cd arcolinux-dotfiles
bash build.sh

```
